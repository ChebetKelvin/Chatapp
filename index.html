WeedShop Navigation Demo

This responsive navigation bar adapts to different screen sizes using Tailwind CSS.

Desktop View

On larger screens, the navigation displays all menu items, search bar, and user actions in a horizontal layout.

Mobile View

On mobile devices, the navigation collapses into a hamburger menu that slides in from the side when activated.

Resize your browser window to see the responsive behavior in action.
